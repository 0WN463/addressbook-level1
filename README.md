# Usage
```sh
c:> java AddressBook addressbook.txt

c:> add John Doe p/3232232 e/johnd@gmail.com 

c:> add Tim Hank p/3434567 e/timh@gmail.com

c:> list
1. John Doe 3232232 johnd@gmail.com 
2. Tim Hank 3434567 timh@gmail.com

c:> delete 2

c:> search john doe
1. John Doe 3232232 johnd@gmail.com

c:> clear
```
